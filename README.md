Este es un proyecto creado en las sesiones de AMAs en Conquer Blocks. 
Este proyecto tiene como objetivo gestionar una red social ( Usuarios, Publicaciones, Estadisticas, Seguidores y Notificaciones) con persistencia en MySql. 

Para poder ejecutar este proyecto necesitas: 
* Python 
* sqlalchemy
* dotenv
* mysql ( workbench)

Para poder conectarte a tu conexion (mySql) necesitas un archivo .env con un formato :
"DB_USER"= ""
"DB_PASSWORD"= ""
"DB_HOST"= ""
"DB_PORT"= ""
"DB_NAME"= ""
